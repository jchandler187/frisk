ClawSec

Security verification for ClawHub skills. 7 checks. 10 intel sources. One verdict.

How it works

ClawSec runs 7 autonomous security checks against any ClawHub skill, backed by 10 continuously-updated threat intelligence sources. Results in under 5 seconds.

🔍 Dependency Scan

Match dependencies against OSV, CISA KEV, and rank by EPSS probability scores.

📜 Static Analysis

Semgrep with 3500+ community rules for vulnerability patterns.

🔑 Secret Scan

Gitleaks detects leaked API keys, tokens, and credentials.

🦠 YARA Scan

Neo23x0 signature-base rules for malware, packers, and suspicious patterns.

🎯 IOC Match

Extract and match URLs, IPs, hashes against URLhaus, ThreatFox, Feodo, MalwareBazaar.

⚠️ Behavioral Heuristics

Detect shell injection, system writes, fetch-exec, and base64 payloads.

🛡️ Prompt Injection

Scan for instruction overrides, role manipulation, and safety bypasses.

CLI

Run locally, no API required.

$ clawsec scan ./my-skill $ clawsec sync $ clawsec status $ clawsec report a1b2c3d4

Install: pip install clawsec or npm i -g @lowwattlabs/clawsec
